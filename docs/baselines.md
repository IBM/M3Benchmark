# M3 Baselines

For the M3 benchmark, we provide users with three baseline models to help get started. Detailed implementations of these baseline models are accessible through the links provided below. Refer to [this page](download_baseline_model_weights.md) for steps to download (and check in) the models weights required for the baseline models.


## Available Baseline Models:

1. [**Qwen-8B Model**](../baselines/qwen-8b.py): For an implementation guide and further details, refer to the Qwen-8B model inline documentation [here](../baselines/qwen-8b.py).
